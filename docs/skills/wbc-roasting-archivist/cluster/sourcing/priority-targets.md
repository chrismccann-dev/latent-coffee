## Frame

The curated per-producer **action axis** ‚Äî "what am I targeting next when I source?" ‚Äî a "soft guiding post" (Chris-locked 2026-05-16; reconciled to canon 2026-06-19), NOT a canonical-validation surface. It is the registry's `ProducerEntry.sourcingPriority` bucket field and drives the `/producers` **Priority targets** tab. It **retired the old Tier 1/2/3 priority-targets framing** (which collided with the producer Reference-role tier and the registry `tier`); the five buckets are the successor. See [docs/reference/wbc-materials.md ¬ß Sourcing priority](docs/reference/wbc-materials.md) for the canonical glossary entry and [docs/taxonomies/producers.md ¬ß Sourcing priority](docs/taxonomies/producers.md) for the live roster.

| Bucket | Shape | Buy logic |
|---|---|---|
| **`pursue`** | Apex-aligned "act when it surfaces" | A standing recognition set ‚Äî engineered process signature + layered-evolving output; the index is pre-computed homework so a bare name + a cup score is enough to act (was T1 Best Fit) |
| **`watch`** | "Strong fit, but unproven / no access" | Portability-exploration lane; engineered fit without evidence or a reachable channel yet (absorbs much of the old T3 Selective) |
| **`learning`** | "Buy when the lot has a job" | Process / variety / roasting-basics build-skill; a variety-coverage gap routes here, not to `pursue` (was T2 High Learning Value) |
| **`calibration`** | "Keep the clean skill sharp" | A fixed clean benchmark (washed Panama Gesha the yardstick) bought at a relaxed cadence ‚Äî the Skill-maintenance lane in bucket form (renamed from `reference`) |
| **`avoid`** | "Deprioritize / don't index" | Empty of named producers today ‚Äî you don't canonically index a one-dimensional producer as a target |

The three axes are deliberately distinct: **Fit** (the ¬ß 9 filter cascade + ¬ß 0 tells, never a stored number) ¬∑ **Role** (the five portfolio lanes, orthogonal) ¬∑ **Action** (the bucket here).
