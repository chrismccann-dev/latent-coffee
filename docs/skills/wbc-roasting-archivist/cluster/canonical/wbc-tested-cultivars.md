## Purpose

This file is the canonical registry of which **WBC-validated cultivar / process / sourcing patterns** have been tested in Latent's own roasting lineup. Treats the WBC corpus as a sourcing palette and tracks adoption rate.

## Schema (to be defined)

For each tested pattern, capture:

- Pattern kind — cultivar / process / sourcing channel / portfolio lane / etc.
- WBC anchor — which competitors / years validated it
- The Latent `green_beans.id` that tested it (FK to green_beans table)
- Outcome — did the pattern translate to a reference roast? Any deviation?
- Carry-forward — did the result inform a Latent canonical pattern, or stay anecdotal?
