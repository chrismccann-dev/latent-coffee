## Cross-lot framing — what this lot taught the Historian

Sudan Rume Washed is the **most data-rich lot in the archive (20+ batches, 6 experiment sets)** and the empirical anchor for several confirmed patterns:

1. **FC floor + drop ceiling tight on high-density washed Colombian (~200°C / ~208°C)** — the canonical evidence row in [cross-coffee-insights.md § FC Floor & Ceiling by Processing Method](../patterns/cross-coffee-insights.md#fc-floor--ceiling-by-processing-method) (High confidence).
2. **WB-to-Ground delta tightens with profile resolution** — #119 closed at delta 1.0 vs V1/V2 typicals 3-5; tight-delta-tracks-cup-quality is now a Confirmed Pattern.
3. **xbloom evaluation recipe under-extracts delicate aromatic washed coffees** — Sudan Rume Washed surfaced this gap and triggered the canonical "optimized brew session before declaring resolved" protocol. Lever: 91°C / 1:14 (vs 94°C / 1:17 evaluation default).
4. **Producer-notes-include-"lemongrass"/"jasmine"/"bergamot" predicts under-extraction at default recipe** — confirmed via the optimized brew session reveal; now lives in [§ Green Spec → Starting Hypothesis](../patterns/cross-coffee-insights.md#green-spec--starting-hypothesis).
5. **Varietal aromatic vocabulary shared across washed + natural** — "fermented / funky" descriptors on this variety usually equal underdevelopment, not defect. The Special Guests (London) direct-comparison session was the unlock; before that, the lemongrass/ginger compounds read as defects.

The varietal aromatic fingerprint lives in [cross-coffee-insights.md § Varietal Aromatic Fingerprints](../patterns/cross-coffee-insights.md#varietal-aromatic-fingerprints).
