## Cross-lot framing — what this lot taught the Historian

The **RoR-shape-not-dev-time** lesson surfaced here is one of the foundational pre-counterflow lessons that carries forward to the counterflow era. The small bridge hold at peak (Batch 63 in classic mode) introduced citrus for the first time after several flat-cup experiments; counterflow later resolved the same issue more efficiently via inlet curve shape.

The Java varietal aromatic fingerprint (clean / structured / balanced when well-developed; thin / cereal when under; roasty / baked when over) lives in [cross-coffee-insights.md § Varietal Aromatic Fingerprints](../patterns/cross-coffee-insights.md#varietal-aromatic-fingerprints).
