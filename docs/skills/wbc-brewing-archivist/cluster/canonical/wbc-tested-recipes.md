# Canonical: WBC-Tested Recipes

**Status:** PLACEHOLDER (Wave 2, 2026-05-26) — sub-resource of the WBC Brewing Archivist cluster per the "knowledge cluster sub-resources" pattern (Chris's Round 2 brainstorm framing).

## Purpose

This file is the canonical registry of which WBC competitor recipes have been **directly tested by Chris on a Latent brew**, vs. which remain reference-only in [wbc-recipes.md](../wbc-recipes.md). Treats the 102-recipe corpus as an aspirational palette and tracks adoption rate.

## Schema (to be defined)

For each tested recipe, capture:

- WBC competitor + year (e.g. "Wölfl 2024")
- The Latent `brews.id` that tested it (FK to brews table)
- Outcome — did the recipe-coffee match work? Any deviation Chris noted?
- Carry-forward — did the result inform a Latent canonical pattern, or stay anecdotal?
